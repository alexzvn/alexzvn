## Zutaten

### Voraussetzungen
- JM Timer installiert (über den Launcher)
- Optional: zweiter Monitor für das Sprecherfenster
- Optional: Netzwerk / Bitfocus Companion für die Fernsteuerung

## Schritt-für-Schritt

### Einrichtung
- Timer-Werte bzw. Presets je Segment anlegen
- Sprecherfenster auf den Talent-Monitor schieben (Vollbild)
- Optional: mit JM Rundown koppeln (ein GO startet den Timer) oder Companion verbinden

### Während
- Timer starten / stoppen / zurücksetzen
- Bei Bedarf eine Kurznachricht für die Bühne einblenden
- Over-Time (Rotphase) im Blick behalten

### Nachbereitung
- Timer für das nächste Segment zurücksetzen

## Profi-Tipps
- Mit JM Rundown koppeln: pro Segment startet der Timer automatisch.
- Den Countdown zusätzlich auf JM Stage Display für die Crew anzeigen.

## Pannenhilfe

| Risiko | Gegenmaßnahme |
| --- | --- |
| Sprecherfenster auf falschem Monitor | Fenster auf den Talent-Monitor ziehen, Vollbild setzen |
| Fernsteuerung reagiert nicht | Gleiches LAN prüfen, Companion-Verbindung kontrollieren |

## Checklisten

### Einrichtung
- [ ] Presets je Segment angelegt
- [ ] Sprecherfenster auf Talent-Monitor
- [ ] Fernsteuerung (optional) verbunden

### Vor Live
- [ ] Test-Start gelaufen
- [ ] Auf Stage Display sichtbar
